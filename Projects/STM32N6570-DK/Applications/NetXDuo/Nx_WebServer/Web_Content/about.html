Product Overview

The STM32N6570-DK Discovery kit is a complete demonstration and development platform for the STMicroelectronics Arm Cortex-M55 core-based STM32N657X0H3Q microcontroller. Arm 32-bit Cortex-M55, 3360 CoreMark, frequency up to 800 MHz, 1280 DMIPS, 32-Kbyte ICACHE, 32-Kbyte DCACHE. Arm MVE (M-Profile vector extension) and Arm TrustZone MPU, NVIC. Single and half-precision floating point unit.

All Features

  • Neural MCU, Arm Cortex-M55, 600-Gops ST Neural-ART Accelerator, Neo-Chrom 2.5D GPU, 4.2 MB contiguous SRAM, in VFBGA264 package
  • 5" capacitive touch LCD display module
  • 1 USB Type-C with USB 2.0 HS interface, DRP
  • 1 USB Type-A with USB 2.0 HS interface, host, 0.5A max
  • 1-Gbit Ethernet compliant with IEEE-802.3-2002
  • SAI Audio CODEC
  • 1 digital microphone
  • 1-Gbit Octo-SPI Flash
  • 256-Mbit Hexa-SPI PSRAM
  • 2 user LEDs
  • 2 User, 1 Tamp and 1 Reset push-buttons
  • ARDUINO UNO R3
  • ST MOD+
  • CSI Camera
  • SD card
  • Board connectors:
    • SD card
    • USB ST-LINK Micro-C
    • USB Type-A
    • Ethernet RJ45
    • Camera FPC connector
    • LCD connector
    • Stereo headset jack including analog microphone input
    • Extension connectors for digital microphones
    • Extension connectors for compatibility with Arduino Uno V3
    • Extension connectors for STmod+
    • STLINK-V3E USB connector
    • USB Type-C connector
    • 5V delivered by RJ45 (Power over Ethernet)
    • USB charger
  • On-board STLINK-V3EC debugger/programmer with USB re-enumeration capability: mass storage, Virtual COM port, and debug port
  • Comprehensive free software libraries and examples available with the STM32CubeN6 MCU Package
  • Support of a wide choice of Integrated Development Environments (IDEs) including IAR Embedded Workbench, MDK-ARM, and STM32CubeIDE
  • Handled by STM32CubeMonitor-UCPD (STM32CubeMonUCPD) software tool

About the demo

this demo is a WebServer running on the STM32N6570-DK board using the NetXDuo API. The WebServer offers 2 webpages.

  • about.html : this page giving an overall description of the demo and the STM32N6570-DK HW features
  • dashboard.html a simple dashboard interface to let the user:
    • Monitor some performance infos collected from the ThreadX and NetXDuo. These infos are updated each 3 seconds via JQuery AJAX requests.
    • Perform a simple action on the board, that is toggling the green led, using a toggle switch button.

Note: To correctly display the webpage, the web browser should be connected to the internet.
