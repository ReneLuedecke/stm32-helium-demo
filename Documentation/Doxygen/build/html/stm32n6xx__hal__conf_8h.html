Set below the peripheral configuration to "1U" to add the support of HAL callback registration/unregistration feature for the HAL driver(s). This allows user application to provide specific callback functions thanks to HAL_PPP_RegisterCallback() rather than overwriting the default weak callback functions (see each stm32n6xx_hal_ppp.h file for possible callback identifiers defined in HAL_PPP_CallbackIDTypeDef for each PPP peripheral).
